1. Gehe zu Google Developers Console und verbinde dich mit deinem Google Account
  2. Klicke Projekt erstellen und gib einen Namen an
  3. Wenn der Vorgang abgeschlossen ist, klicke auf das neu erstellte Projekt
  4. Gehe zu APIS & AUTH / Zugangsdaten
  5. Click Neue Client ID erstellen
  6. Geben sie erlaubte Domain als Authorized Javascript origins an und die oben genannte Callback URL als Authorized redirect URI
  7. Jetzt kopieren Sie die Client ID und das Client Secret in die Google2Piwigo Konfiguration
  8. Fertig!
